# word2vec-numpy
This project is part of the JetBrains internship application. The model that I have implemented is the Skip-gram model with negative sampling based on the following papers: https://arxiv.org/abs/1301.3781 https://arxiv.org/abs/1310.4546.

Trained on a subset of the Wikitext2 dataset under the CC-BY-SA 4.0 license, available [here](https://huggingface.co/datasets/Salesforce/wikitext).

The model weights are available in releases.
